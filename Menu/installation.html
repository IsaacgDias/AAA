INSTALLATION MANAGER

Você pode usar o Gerente de Instalação do Guardium ® (GIM) para instalar e manter componentes do Guardium em servidores gerenciados. O componente GIM inclui um servidor GIM, que é instalado como parte do sistema Guardium , e um cliente GIM, que deve ser instalado em servidores que hospedam bancos de dados ou sistemas de arquivos que você deseja monitorar. O cliente GIM é um conjunto de scripts Perl que executam em cada servidor gerenciado. Depois de instalar o cliente GIM, ele funciona com o servidor GIM para executar estas tarefas:

- Confira as atualizações para o software instalado

- Transferir e instalar novo software

- Desinstalar software

- Atualizar parâmetros de software

- Monitorar e parar processos que são executados no servidor de banco de dados
