PERSONALIZAÇÃO

As ferramentas Customize Navegação Menu e Customize User / Role permitem que você altere convenientemente o conteúdo e a organização do menu de navegação. Você pode acessar essas ferramentas em vários locais:

- Todos os usuários podem customizar o seu próprio menu de navegação abrindo o menu Usuário no banner Guardium e selecionando Personalizar.

- Os usuários administrativos podem customizar o menu de navegação para outros usuários e funções, abrindo o menu Usuário e selecionando Customizar Usuário / Função ou navegando para Setup > Ferramentas e Visualizações > Customizar Usuário / Função.

- Os usuários logados como accessmgr podem customizar o menu de navegação para outros usuários e funções, navegando para Acesso > Gerenciamento de Acesso, selecionando Navegador De Funçãoe clicando no link Customizar Menu de Navegação.

Após clicar no botão OK , o menu de navegação Guardium é atualizado para refletir quaisquer alterações que você tenha feito na lista Menu Navegação . As seguintes limitações se aplicam ao utilizar estas ferramentas:

- Não é possível excluir o grupo My Dashboards , mas você pode excluir painéis individuais dentro do grupo.

- Novos grupos que estão vazios não serão salvos.

- Os grupos vazios mostrados na lista Menu Navegação não aparecerão no menu de navegação Guardium .
