CONCEITOS E FERRAMENTAS

- Consultas e relatórios: As consultas do Guardium descrevem um conjunto de informações obtidas a partir dos dados coletados. Cada consulta tem um relatório correspondente com o mesmo nome.

- Controle de acesso: O Guardium fornece mapas de acesso como uma forma de mostrar convenientemente o acesso de dados entre clientes de banco de dados e servidores de banco de dados.

- Funções do usuário: Uma função define um grupo de usuários do Guardium que compartilham os mesmos privilégios de acesso.

- Grupos: O Guardium suporta o agrupamento de elementos para simplificar a criação e o gerenciamento de políticas e para esclarecer a apresentação de relatórios.

- Arquivo de dados e purga: Arquivo de dados faz backup de dados que foram capturados por um sistema Guardium. Ao configurar o Data Archive, os critérios de purga de dados também podem ser especificados.

- Guardio Installation Manager: O Guardium Installation Manager (GIM) é usado para instalar e manter componentes do Guardium em sistemas gerenciados.
